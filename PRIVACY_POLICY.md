# Privacy Policy for Always show scrollbar

Last updated: May 16, 2026

Always show scrollbar is a browser extension that keeps native page scrollbars visible on sites selected by the user.

## Open source policy

Always show scrollbar is fully open source under the MIT License. The extension source code, documentation, UI files, localization files, generated store assets, and included logo assets are available under that license unless a file explicitly says otherwise.

The open-source license does not change the privacy behavior described below: the extension does not collect or transmit user data.

## Data the extension stores

The extension stores only local extension settings:

- enabled domain rules, such as `*`, `example.com`, or `file://`;
- selected interface language.

These settings are saved with the Chrome `storage.sync` API so they can sync through the user's browser profile when Chrome Sync is enabled.

## Data collection

The extension does not collect, sell, transmit, or share personal data with the developer or any third party.

The extension does not use analytics, advertising SDKs, tracking scripts, remote code, external APIs, or telemetry endpoints.

## Website access

The extension needs access to webpages so it can apply CSS fixes that keep scrollbars visible on enabled sites. It does not read page text for profiling, does not log browsing history, and does not send visited URLs to any server.
